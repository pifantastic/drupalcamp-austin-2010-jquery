the basics

$() <- wtf is this?

  • It's a function!
  • It's an alias for jQuery
  • Use them interchangeably*
$ === jQuery; // true
* Be warned! Some other javascript libraries also use $ and jQuery's $ may get overwritten!

the basics

What does it do?

  • $('selector') it takes a *CSS3 selector
  • and returns an Array-like Object full of delicious DOM elements
var list_items = $('li'); for (var x = 0; x < list_items.length; x++) { alert(list_items[x]); }
* jQuery actually supports more than are defined in CSS 1-3 http://api.jquery.com/category/selectors/
