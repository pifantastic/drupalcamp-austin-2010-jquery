the basics

F'ing $, how does it work?

  • It's a function! $()

  • It's an alias for jQuery()

  • Use them interchangeably*

$ === jQuery; // true
* Be warned! Some other javascript libraries also use $ and jQuery's $ may get overwritten if those libraries are also present on the page.

the basics

What does it do?

  • takes a CSS3* selector, like 'li' or '.hello' or '#thing'
  • returns an Array-like Object full of delicious DOM elements
  • usually referred to as a jQuery object
var list_items = $('li');
* jQuery actually supports more than are defined in CSS 1-3 http://api.jquery.com/category/selectors/

Chaining

jQuery lets us chain things together

$('#foo') // Find the element with id=foo .hide() // Hide it .css('color', 'yellow') // Change its text color to yellow .show() // Show it .find('a') // Find all links that are children of #foo .append('<span>hi!</span>') // Append some HTML to each of those links .css('background-color', 'aqua'); // Change their background color to aqua

Event binding

jQuery makes it dead simple to bind events to elements

Event binding

binding to a link's click event

clicky -> Google.com

$('a').click(function() { alert('You are about to go to ' + $(this).attr("href")); });

Event binding

disabling the browser's default behavior

clicky -> Google.com

$('a').click(function(e) { alert("Don't even think about going to " + $(this).attr("href")); e.preventDefault(); });

Event binding

stopping an event

Google.com

<p><a href="http://google.com">Google.com</a></p> $('a').click(function(e) { if (confirm('Stop propagation?')) { e.stopPropagation(); } }); $('p').click(function(e) { alert("Thanks for bubbling to me!"); });

Event binding

stopping an event and preventing the default behavior

Google.com

<p><a href="http://google.com">Google.com</a></p> $('a').click(function(e) { return false; });

Animation

jQuery makes it easy to do simple animations:

  • show/hide elements

  • slide up/down elements

  • fade in/out elements

Animation

show/hide an element

/
$('#show').click(function() { $('.red-box').show(); }); $('#hide').click(function() { $('.red-box').hide(); });

Animation

slide up/down an element

/
$('#slideup').click(function() { $('.red-box').slideUp(); }); $('#slidedown').click(function() { $('.red-box').slideDown(); });

Animation

fade in/out an element

/
$('#fadein').click(function() { $('.red-box').fadeIn("fast", function() { alert('Nice fadein bro.'); }); }); $('#fadeout').click(function() { $('.red-box').fadeOut("slow"); });
